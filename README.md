# Portfolio-simple-website

## Portfolio Website

A personal portfolio website built using **HTML** and **CSS**. The website consists of two pages: a **Home** page and a **Contact** page. It is designed with a clean and simple user interface to provide information about me and make it easy for visitors to get in touch.

### Features

* Home page with personal introduction
* Contact page for communication details
* Simple and responsive design
* Easy navigation between pages
* Built using basic HTML and CSS

### Technologies Used

* HTML5
* CSS3

### Purpose

This project was created to practice the fundamentals of web development, including webpage structure, styling, and navigation using HTML and CSS.

**Short Description (for the repository description field):**

> A simple personal portfolio website built with HTML and CSS, featuring Home and Contact pages with a clean and responsive design.
